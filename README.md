# Fabricate (Fab for short)

Fabricate is a meta buildsystem with the purpose of giving you (the programmer) as much control over the build process as possible. Fabricate uses lua as its configuration language and ninja as the backend.

## Usage

The usage is outlined under the chariot `--help` option.

## Configuration

Work in progress. See an example at [cronus](https://github.com/elysium-os/cronus) kernel repo. The globals exposed to lua are outlined in the [definitions](./src/lua/definitions.d.lua) file.
